(1).容器拷贝到宿主机

(2).宿主机拷贝到容器内

3.dockerfile中COPY和ADD命令区别

1
2

相同点: COPY和ADD指令都可以讲主机上的资源复制到容器镜像中
不同点: copy指令只能从所在宿主机读取资源，ADD还支持通过URL从远程服务器读取资源，另外ADD可以直接读取本地tar文件并解压
